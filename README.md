# Skill-Match Job Portal

## Description

Skill-Match Job Portal is an AI-driven job platform designed to match candidates with job opportunities based on their skills rather than just qualifications. By focusing on skill sets, the platform aims to make job searches more inclusive and efficient.

## Features

- **AI Skill Matching**: Matches job seekers with relevant jobs based on skills.
- **Profile Creation**: Candidates create profiles with skills and experience.
- **Job Listings**: Employers post jobs emphasizing required skills.
- **Recommendations**: Personalized job suggestions for candidates.
- **Search & Filter**: Search and filter jobs based on skills, experience, and location.

## Technologies Used

- **Frontend**: React.js
- **Backend**: Node.js, Express, MySQL
- **AI/ML**: Python, TensorFlow, scikit-learn, SpaCy (for NLP)
- **Authentication**: JWT
- **Deployment**: Heroku/Netlify

## Setup

### Prerequisites

- Node.js
- MySQL
- Python 3.x (for AI model)
- npm or yarn

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/skill-match-job-portal.git
    cd skill-match-job-portal
    ```

2. Install frontend dependencies:
    ```bash
    cd frontend
    npm install
    ```

3. Install backend dependencies:
    ```bash
    cd backend
    npm install
    ```

4. Set up your database and configure your `.env` file for DB credentials.

5. Run the project:
    ```bash
    npm start
    ```

6. Visit `http://localhost:3000` to access the portal.

## Tests

Unit tests can be run using **Jest**:
